# Vidly

## Description

Vidly is a video rental app

## Getting Started

### Installing

- Clone this repository to your local machine
- Install dependencies by running:

```
npm install
```

or

```
yarn
```

### Starting the server

- Run below command to start the dev server

```
npm start
```

or

```
yarn start
```
